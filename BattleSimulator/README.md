# 战斗模拟系统

这是一个用C#开发的功能完整的战斗模拟系统，可以模拟不同角色或实体之间的战斗过程。

## 系统架构

系统由以下几个核心模块组成：

1. **核心模型**：定义了实体、技能和状态效果的基本属性和行为
2. **角色类型**：实现了不同类型的角色，如战士和法师
3. **战斗系统**：负责控制战斗流程、执行战斗逻辑和判定战斗结果
4. **测试项目**：提供了一个演示程序，展示系统的功能

## 主要功能

- 角色属性管理（生命值、攻击力、防御力等）
- 多样化的技能系统（伤害、治疗、增益、减益等效果）
- 丰富的状态效果（中毒、眩晕、攻击力提升等）
- 完整的战斗流程控制（回合制战斗机制）
- 智能的战斗结果判定
- 可扩展的系统架构（便于添加新角色类型、技能或战斗规则）

## 如何使用

1. 打开解决方案文件 `BattleSimulator.sln`
2. 生成解决方案（Build -> Build Solution）
3. 运行测试项目 `BattleSimulator.Tests`
4. 按照控制台提示进行操作，观看战斗模拟过程

## 系统特点

- **模块化设计**：各个组件之间职责明确，耦合度低
- **可扩展性强**：通过继承基类可以轻松添加新的角色类型、技能或状态效果
- **战斗逻辑丰富**：包含了命中率、暴击率、闪避率等多种战斗元素
- **可视化战斗过程**：通过控制台界面展示战斗的每一个回合和角色状态

## 扩展指南

### 添加新角色类型

1. 创建一个继承自 `Entity` 的新类
2. 设置角色的基础属性
3. 初始化角色的特有技能
4. 可以重写 `TakeDamage` 等方法来实现角色的特殊能力

### 添加新技能

1. 创建一个继承自 `Skill` 的新类
2. 设置技能的基本信息（名称、描述、消耗、冷却等）
3. 重写 `ApplyEffect` 或相关方法来实现技能的特殊效果

### 添加新状态效果

1. 创建一个继承自 `StatusEffect` 的新类
2. 实现 `Apply`、`Update`、`OnExpire` 和 `Clone` 方法
3. 设置状态效果的基本信息（名称、描述、持续时间等）
